Overview

ID3COM is a collection of Visual Basic compatible Component Object Model (COM) (also know as ActiveX) classes for manipulating ID3v2 tags in MPEG audio files. ID3v2 is a standard way for embedding descriptive information within MPEG audio files. The ID3v2 standard is documented at http://www.id3.org. ID3COM makes dealing with ID3 tags easy.

You can always find the latest version of ID3COM at http://www.kindel.com/mp3/id3com.

ID3COM was developed to help foster the usage of ID3v2 tagging of MPEG II Layer 3 (MP3) audio files by providing a simple interface for developers using high-level programming environments. ID3COM was designed to very “VB friendly” and its object model conforms to the standards set by Microsoft for Visual Basic components. However, ID3COM can be used from any programming language that supports COM, including Visual J++, Inprise’s (Borland’s) Delphi, and C/C++.

It is expected that programmers will find ID3COM useful in GUI applications, scripting, and server scenarios. ID3COM is implemented in a thread-safe way, and supports COM’s “Free Threaded” model allowing efficient execution when running on a server (e.g. within an Active Server Pages application).

Licensing

ID3COM is freeware. You may use it within your applications at no cost. However, if you re-distribute ID3COM with your application you must follow a few simple rules.

ID3COM is distributed by Kindel Systems as a self-installing executable package (ID3COM.EXE). ID3COM.EXE must not be re-distributed with your applications. If you wish to re-distribute the ID3COM objects with your application you may only re-distribute ID3COM.DLL and you must follow the rules for Win32 file version checking when installing and un-installing ID3COM.DLL

If you re-distribute ID3COM.DLL with your application you must register your application at http://www.kindel.com/mp3/id3com/register.asp.  Other than having to register your usage of ID3COM, there is no cost associated with re-distribution.

Users Guide

Using ID3COM From Visual Basic

To use ID3COM from Visual Basic you need to add ID3COM as a reference to your VB project. To do this:

·         Choose References from the Project menu.

·         Find “Kindel Systems ID3COM Library” in the list of Available References and check the checkbox next to it. Hit OK to close the dialog.

After doing this you will be able to access the ID3COM classes and definitions using the normal VB mechanisms. For example, to create a new instance of the ID3Tag class you would do:

Dim tag As New ID3Tag

 

Or

Dim tag As ID3Tag

Set tag = New ID3Tag

 

You can also use the Visual Basic Object Browser to view the ID3COM object model.

Object Model Reference

ID3COM defines the following object model:

ID3Tag

     ID3Frames

          ID3Frame

               ID3Fields

                        ID3TextItems

     ID3VersInfo

Of these classes, only ID3Tag and ID3Frame can be created directly.

Below is a detailed reference for the ID3COM object model. Each section includes a description of the class, an overview of the class’ methods and properties, and details for each method or property. Examples are provided throughout.

ID3Tag Class

This class represents an ID3 tag within an audio file. A given file may have at most one ID3 Tag.

The ID3Tag class serves as a container class for all of the ID3 frames in the tag. You can access the collection of frames (represented as ID3Frame objects) using the Frames property.

ID3Tag can be used for ID3 tags that are stored either in memory or attached to audio files. The IsTagHeader, Size, Parse, and Render methods are useful when manipulating tags in memory buffers. The Attach and Update methods are useful when reading and writing tags from files.

ID3Tag is a class object meaning that new instances can be created using Visual Basic’s New operator (Dim tag As New ID3Tag), Visual Basic’s CreateObject function (Set tag = CreateObject(“Kindel.ID3Tag”), and COM’s CoCreateInstance function (use CLSID_ID3Tag defined in id3com.h).

Examples

 

Adding a tag to an existing MP3 file in VB

Dim tag As ID3Tag

Set tag = New ID3Tag

tag.Link Filename := “Some Great Song.mp3”, CreateFile := False

tag.Album = “A Great Album”

tag.Artist = “A Great Band”

tag.TrackTitle = “Some Great Song”

tag.TrackNum = 3

tag.Update

 

Adding a tag to an existing MP3 file in VB

Dim tag

Set tag = CreateObject(“Kindel.ID3Tag”)

tag.Attach “Some Great Song.mp3”, False

tag.Album = “A Great Album”

tag.Artist = “A Great Band”

tag.TrackTitle = “Some Great Song”

tag.TrackNum = 3

tag.Update

 

Methods and Properties

Sub Attach(Filename As String, [CreateFile As Boolean]) As Long

Attaches the ID3Tag object to the file specified. Returns the offset (in bytes) in the file where the tag ends and the audio starts.

If the file specified does not exist and the optional parameter CreateFile is True the file will be created. If the file specified does not exist and CreateFile is False the file will not be created and an error will be thrown.

CreateFile is True by default.

Sub Update()

Saves the tag to the file that was specified with Link.

Sub Clear()

Initializes the ID3Tag object.

Property Title As String

Sets or retrieves the value of the Title (TIT2) frame from the tag.

Property Album As String

Sets or retrieves the value of the Album (TALB) frame from the tag.

Property Artist As String

Sets or retrieves the value of the Artist (TPE1) frame from the tag.

Property Year As String

Sets or retrieves the value of the Year (TYER) frame from the tag.

Property Genre As String

Sets or retrieves the value of the Genre (TCON) frame from the tag.

Property Comment As String

Sets or retrieves the value of the Comment (COMM) frame from the tag.

Function IsTagHeader(Header) As Long

Gets the number of extra bytes in the rest of the tag. Header is an array of bytes.

Sub Parse(Header, Buffer)

Parse binary data into ID3 frames. Header is an array of bytes containing the header and Buffer is an array of bytes containing the rest of the tag.

Sub Render(Buffer)

Renders the tag to an array of bytes.

Property Size As Long

Returns an over estimate of the number of bytes required to store a binary version of the tag.

Property Frames As ID3Frames

Returns the ID3Frames collection for the tag.

Property HasChanged As Boolean

Indicates whether the tag has changed since the last parse.

Property UnSync As Boolean

Tells the object to unsync the binary tag.

Property ExtendedHeader As Boolean)

Tells the objct to create an extended header when rendering.

Property Compression As Boolean

Tell the object whether to compress frames or not.

Property Padding  As Boolean

Tells the object to create padding.

ID3Frames Class

The ID3Frames class is a collection of ID3Frame objects. The ID3Tag class supports a Frames property which returns an ID3Frames object.

You can use the Visual Basic For…Each control structure to enumerate over the ID3Frame objects in an ID3Frames collection.

Methods and Properties

Function Item(Item, [FieldID As ID3LibFieldID], [Data]) As ID3Frame

Retrieves a frame from the tag by index or 4 character frame ID

Property Count As Long

Returns the number of frames in the tag.

Sub Add(frame As ID3Frame)

Adds a frame to the tag.

Sub Remove(frame As ID3Frame)

Removes the specified frame from the tag.

ID3Frame Class

The ID3Frame class represents a single frame within an ID3Tag. You can create new instances of ID3Frame objects and them insert them into a tag using the Add method of the ID3Frames collection.

ID3Fields Class

The ID3Fields class is a collection of ID3Field objects. The ID3Frame class supports a Fields property which returns an ID3Fields object.

You can use the Visual Basic For…Each control structure to enumerate over the ID3Field objects in an ID3Fields collection.

Methods and Properties

Property Count As Long

Returns the number of fields in the frame.

ID3Field Class

The ID3Field class represents a field within an ID3Frame object. Each ID3Frame type (identified by it’s four character ID) has a fixed number of fields specified by the ID3v2 specification. You can use the Fields property of the ID3Frame class to access the collection of fields within a frame.

Methods and Properties

Property Value

The value of the field.

Property ID As ID3LibFieldID

ID3Lib Field ID. Read-only.

Sub Clear()

Initializes the field.

Property Size As Long

The size (in bytes) of the field.

Sub FromFile(Filename As String)

Copies data from a file into the field.

Sub ToFile(Filename As String)

Saves the data in the field to a file.

Property TextItems As ID3TextItems

Returns the text items collection for the field.

ID3TextItems Class

The ID3TextItems class is a collection of Strings. The ID3Field class supports a TextItems property which returns an ID3TextItems object.

You can use the Visual Basic For…Each control structure to enumerate over the Strings in an ID3TextItems collection.

Methods and Properties
